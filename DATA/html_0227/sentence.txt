→System 名前空間を参照する using ディレクティブで始まる。
 (名前空間は、プログラムとライブラリを階層的に整理するための手段）・class Hello
→class名は、Hello（Java : public class Hello）・static void Main( )

→出力（Java : System.out.println("Hello, World");）・値型と参照型の2種類の型がある。
値型の変数が直接データを格納するのに対して、参照型の変数はデータへの参照を格納する。出力時、Console.WriteLine(　);だと改行される。
Console.Write(　);だと改行されずに引き続き出力される。・string
→文字列を定義する。（Java : String）・int型は、整数を表す。
　+ : 加算
　- : 減算
　* : 乗算
　/ : 除算・int型には最小値と最大値の制限がある。
これらの制限を超える値が作られると、オーバーフローの状態になる。・ifステートメントを使用した条件判断・if と else を組み合わせ・ループを使用した処理の繰り返し
whileステートメントは、条件が false になるまで、条件の確認と実行を繰り返す。・for ループの処理簡単にまとめはしましたが、ほとんどJavaと同じような感じになりました。
