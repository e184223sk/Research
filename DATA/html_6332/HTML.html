WindowsC#WPF.NETFramework

C# + WPF(Windows Presentation Foundation) の開発に関わる便利ツール、情報のまとめ

この記事について

WPFでデスクトップアプリケーションを作成する目的として、関連するC#知識とライブラリの情報をまとめた記事になります。本記事の内容はUWP/Xamarin/Unity環境では異なる点が多いため注意してください。

マルチプラットフォーム対応が視野にある場合はWPFではそのままの移植が難しいため、UWP/Xamarin/Unity環境で実現が可能かの検討をすることをおすすめします。(.NET Core環境では現状動作しませんし今後対応される予定もなさそうなため)

パフォーマンス

WPFのデータバインディングは便利ですが、速度が要求される部分になるとチューニングが必要になります。

WPFではUIの仮想化が一部の標準のコントロール(WrapPanel,Canvas等)だとできないので注意が必要です。

MVVMフレームワーク

MVVM Light Toolkit

ViewModelLocaterとMessanger機能が使いたい人向けMVVMフレームワーク。
今だとReactivePropertyと組み合わせて使うのをおすすめします。

  • 本家サイト：MVVM Light Toolkit
  • GitHub：lbugnion/mvvmlight

機能

  • ViewModelLocaterを利用した View <=> ViewModel の紐付け (Prismと違って自動紐付け機能はない簡易版)
  • Messanger 機能のサポート (ViewModelBaseを継承したオブジェクト同士のみ)
  • RelayCommandによるコマンド実装補助（これはReactiveProperyのReactiveCommandのほうがよい）

使用感としては小規模なC#プロダクトでは十分かとは思いますが、ある程度大きくなるとViewModelLocaterが肥大化してきて困るかもしれません。その場合はPrismの方に乗り換えたほうが良いかもです。

ドキュメント

日本語ドキュメントはあんまり無いですが、機能も最低限なのであまり困らないかとは思います。

  • MVVMパターンのシンプルなサンプル（MVVM Light + ReactiveProperty) 言語: C#, XAML Visual Studio 2013 用
  • 結局最後は「MVVM + Messenger + Behavior パターン」に行き着いた - present

ChainingAssertion

neuecc/ChainingAssertion: Method Chaining base UnitTesting Extension Methods and Dynamic Private Accessor for MSTest, NUnit, xUnit.net.

テスト時にいろいろと便利な拡張メソッドを定義してくれるパッケージ。

  • MSTest, NUnit, MbUnit, xUnitに対応。
  • 一般的な Assert.AreEqual(A, B) などの代わりに、A.Is(B) などのアサートを追加することが出来る。
  • MSTestでxUnitのようにParameterized Testを実行できる拡張を提供

概要

neue cc - テストを簡単にするほんの少しの拡張メソッド
neue cc - Chaining Assertion for MSTest

使用時の注意点

Parameterized Test, TestCaseSource使用時には public TestContext TestContext { get; set; } の定義が必須になります。
（こちらを定義しないとTextContext.Run()が存在しないというエラーで散々悩む羽目に・・・）
https://github.com/neuecc/ChainingAssertion#parameterized-test-for-mstest

[TestClass]
public class UnitTest
{
    public TestContext TestContext { get; set; } //必須

    [TestMethod]
    [TestCase(1, 2, 3)]
    [TestCase(10, 20, 30)]
    [TestCase(100, 200, 300)]
    public void TestMethod2()
    {
        TestContext.Run((int x, int y, int z) =>
        {
            (x + y).Is(z);
            (x + y + z).Is(i => i < 1000);
        });
    }
}
