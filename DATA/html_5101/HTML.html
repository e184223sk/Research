【Unity】UnityAdsで広告表示, 広告を一定周期で再表示

C#Unityゲーム制作

ゲームやリワード内容にもよりますが、リワード広告は表示する周期によってリテンション率が変わると言われてます。

そこで以下のような機能を実装する方法を紹介します。
・ボタンを押したら広告が表示される
・見終わると広告ボタンが非表示になる
・一定時間が経つと再度ボタンが表示する

Script

UnityAdsCtr.cs
using UnityEngine;
using UnityEngine.Advertisements;
using UnityEngine.Events;
using System;

/// <summary>
/// 広告表示とButton再表示を制御する
/// 常にActiveなGameObjectにアタッチして使用する
/// </summary>
public class UnityAdsCtr : MonoBehaviour
{
    //表示周期(編集可能)
    [SerializeField] int DisplaySpan = 10;
    TimeSpan lastTime;
    TimeSpan NowTime;
    //表示ボタン
    [SerializeField] GameObject AdsButton;
    //リワード処理
    public UnityEvent _UnityEvent;

    /// <summary>
    /// リワード広告を表示する
    /// Buttonイベントに登録するなどして実行する
    /// </summary>
    public void ShowRewardedAd()
    {
        if (_UnityEvent == null)
        {
            return;
        }
        if (Advertisement.IsReady("rewardedVideo"))
        {
            var options = new ShowOptions { resultCallback = HandleShowResult };
            Advertisement.Show("rewardedVideo", options);
        }
    }

    /// <summary>
    /// 広告を全て見るとリワード報酬を獲得する
    /// </summary>
    /// <param name="result">Result.</param>
    private void HandleShowResult(ShowResult result)
    {
        switch (result)
        {
            case ShowResult.Finished:
                Debug.Log("The ad was successfully shown.");
                _UnityEvent.Invoke();
                AdsButton.SetActive(false);
                lastTime = DateTime.UtcNow.TimeOfDay;
                break;
            case ShowResult.Skipped:
                Debug.Log("The ad was skipped before reaching the end.");
                break;
            case ShowResult.Failed:
                Debug.LogError("The ad failed to be shown.");
                break;
        }
    }

    private void Update()
    {
        DisplayCheck();
    }

    /// <summary>
    /// 表示周期が過ぎたらButtonを表示
    /// </summary>
    void DisplayCheck()
    {
        //Activeな時は処理を行わない
        if (AdsButton.activeSelf)
        {
            return;
        }
        NowTime = DateTime.UtcNow.TimeOfDay;
        var span = NowTime - lastTime;
        //Debug.Log("span.TotalMinutes " + span.TotalMinutes);
        if (span.TotalMinutes > DisplaySpan)
        {
            AdsButton.SetActive(true);
        }
    }

    public void Test()
    {
        Debug.Log("Invoke");
    }
}

概要

◆UnityAdsの準備：
こちらを参照。非常に簡単です。
https://unity3d.com/jp/services/ads/quick-start-guide

◆実装例の概要：
・時間経過で宝箱が空く機能があるとします。
・広告を見ると時短できるとします。

◆画面の表示：左上に広告ボタンがあります

◆Hierarcky：AdsSkipButtonが広告表示ボタンです

◆Inspector：
・SceneManagerなど常にActiveなGameObjectにアタッチします(AdsSkipButtonはNonactiveになるので適しません)
・DisplaySpanは分単位です。Scriptではデフォルトが10分で、スクショではテストのため1分にしてます。
・ButtonにEventを登録するように、リワード処理をUnityEventに登録します。スクショではテスト用のメソッドを登録してます。

◆処理結果：
広告視聴後、時間が経過するとボタンが再表示されます。

広告を表示したら広告ボタンが非表示になり、一定時間（動画ではテストのため1分）が経過したらボタンが再表示される機能を実装。 #Unity pic.twitter.com/axjNWVa351
