[VisualStudio] よく書くコードはスニペットにまとめて爆速コーディング

C#VisualStudioコードスニペット

スニペットを作ったらコーディングが捗ったという内容

スニペットとは？

コードの断片
任意のテンプレートを一瞬で呼び出せる機能
VisualStudioには最初からいくつか登録されている

作り方

Visual Studio Snippet Generator
スニペットを簡単に作れるサイト

項目について

項目 内容
Shortcut スニペットを出すのに使う文字
Description スニペットについての説明
Author 作成者
Code 召喚するコード

手順

作成の手順

  1. 項目を埋める
  2. Generateを押す
  3. 下にスニペットが出力されたのを確認する
  4. Downloadを押す

VisualStudioに追加する手順

  1. ツールを選ぶ
  2. コードスニペットマネージャーを選ぶ
  3. コードスニペットマネージャーのインポートを押す
  4. ダウンロードしたスニペットを選択する
  5. 完了を押す

実際に使ってみる

自分の設定したショートカット文字を入力してTabを2回押す

とても便利！

※出てこない人
ツール->オプション->テキストエディター->C#->IntelliSense->スニペットの動作 から好きな設定を選ぶ
