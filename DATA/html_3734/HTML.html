Bit演算のちょっとしたテクニック備忘録。

C#bitOperation

初めに。

あくまで自分用のメモです。
最適化しきれていなかったり、時々間違えたりします。
間違いのご指摘や、何か他に良い方法をご存知でしたら、
コメントの方よろしくお願いいたします。
関数の名称についても、いい名称がありましたらコメント下さると有難いです。

And

Andを返します。 ↑

public static int And(this int num, int n) => num & n;

Or

Orを返します。 ↑

public static int Or(this int num, int n) => num | n;

Xor

Xorを返します。 ↑

public static int Xor(this int num, int n) => num ^ n;

Not

Notを返します。 ↑

public static int Not(this int num) => ~num;

Abs

絶対値を返します。 ↑

public static int Abs(int num) => (num ^ (num >> 31)) - (num >> 31);

Sign

符号を返します。　（負数 => -1,零 => 0,正数 => 1）↑

public static int Sign(int num) => (num >> 31) - (-num >> 31);

Clamp

数値をa以上b以内に制限します。
a < b である必要があります。 ↑

public static int Clamp(int num, int a, int b) => (Abs(num - a) - Abs(num - b) + a + b) / 2;

ClampNonRestriction

数値を a以上b以内 もしくは b以上a以内 に制限します。 ↑

public static int ClampNonRestriction(int num, int a, int b) => (a + b + ((Abs(num - a) - Abs(num - b)) ^ a.GreaterThan(b)) - a.GreaterThan(b)) / 2;

RightmostBit

右端ビットを取得します。 ↑

public static int RightmostBit(int num) => num & -num;

LeftmostBit

左端ビットを取得します。 ↑

public static int LeftmostBit(int num) => (FillLower(num) >> 1) + 1;

RightmostZero

右端のゼロを取得します。 ↑

public static int RightmostZero(int num) => ~num & (num + 1);

FillLower

左端ビットから下位のビットすべてを1で埋めます。 ↑

public static int FillLower(int num)
{
    num |= num >> 1;
    num |= num >> 2;
    num |= num >> 4;
    num |= num >> 8;
    return num | num >> 16;
}

FillHigher

右端ビットから上位のビットすべてを1で埋めます。 ↑

public static int FillHigher(int num)
{
    num |= num << 1;
    num |= num << 2;
    num |= num << 4;
    num |= num << 8;
    return num | num << 16;
}

TrailingZerosMask

最下位ビットから左端ビットまでのマスクを返します。 ↑

public static int TrailingZerosMask(int num) => ~num & (num - 1);

TrailingOnesMask

最下位ビットから連続して立っているビットのマスクを返します。 ↑

public static int TrailingOnesMask(int num) => ~(num + 1) & num;

BitCount

立っているビットの個数(1の個数)を取得します。 ↑

public static int BitCount(int num)
{
    num = (num & 0x55555555) + ((num >> 1) & 0x55555555);
    num = (num & 0x33333333) + ((num >> 2) & 0x33333333);
    num = (num & 0x0F0F0F0F) + ((num >> 4) & 0x0F0F0F0F);
    num = (num & 0x00FF00FF) + ((num >> 8) & 0x00FF00FF);
    return (num & 0x0000FFFF) + ((num >> 16) & 0x0000FFFF);
}

NLZ

左端ビットより上位にある 0 の個数を取得します。 ↑

public static int NLZ(int num) => BitCount(~FillLower(num));

NTZ

右端ビットより下位にある 0 の個数を取得します。 ↑

public static int NTZ(int num) => ((num >> 31) | (-num >> 31)) & BitCount((num & -num) - 1);

Max

より大きい方の数を返します。 ↑

public static int Max(int a, int b)
{
    var mask = a.LessThan(b);
    return (b & mask) | (a & ~mask);
}

Min

より大きい方の数を返します。 ↑

public static int Min(int a, int b)
{
    var mask = a.GreaterThan(b);
    return (b & mask) | (a & ~mask);
}

ShiftL

shiftビット左シフトをします。shiftが32以上のとき、結果は0になります。 ↑

public static int ShiftL(this int num, int shift) => (num << shift) & shift.LessThan(32);

ShiftR

shiftビット左シフトをします。shiftが32以上のとき、結果は0になります。 ↑

public static int ShiftR(this int num, int shift) => (num >> shift) & shift.LessThan(32);

Flag

nビットを立てます。 ↑

public static int Flag(int n) => 1 << n;
public static int SetFlag(this int num, int n) => num | Flag(n);

Mask

mからnビットまでのマスクを立てます。 ↑

public static int Mask(int m, int n) => (Flag(n) - 1) ^ (Flag(m) - 1);
public static int SetMask(this int num, int m, int n) => num | Mask(m, n);

RotateL

shiftビット回転させます。 ↑

public static int RotateL(this int num, int shift) => (num << shift) | (num >> (32 - shift));

RotateL

shiftビット回転させます。 ↑

public static int RotateR(this int num, int shift) => (num >> shift) | (num << (32 - shift));

SetFlags

フラグを立てます。 ↑

public static int SetFlags(this int num, int flags) => num | flags;

ClearFlags

フラグを消します。 ↑

public static int ClearFlags(this int num, int flags) => num & ~flags;

ToggleFlags

フラグを切り替えます。 ↑

public static int ToggleFlags(this int num, int flags) => num ^ flags;

HasFlags

フラグが全て立っていれば-1そうでなければ0を返します。 ↑

public static int HasFlags(this int num, int flags) => ((num & flags) ^ flags).EqualTo(0);

HasAnyFlags

フラグが全て立っていなければ0そうでなければ-1を返します。 ↑

public static int HasAnyFlag(this int num, int flags) => ((num & flags) ^ flags).InequalTo(0);

HasJustFlags

指定したフラグのみが全て立っていれば-1そうでなければ0を返します。 ↑

public static int HasJustFlags(this int num, int flags) => (num ^ flags).EqualTo(0);

IntLog2

2を底とした対数の切り捨てを返します。System.Math.ILogBが同等です。 ↑

public static int IntLog2(int num) => BitCount(FillLower(num)) - 1;

NextPowerOfTwo

より大きな2の冪を返します。 ↑

public static int NextPowerOfTwo(int num)
{
    if (num <= 0) return 1;
    return FillLower(num) + 1;
}

PrevPowerOfTwo

num以下の2の冪を返します。 ↑

public static int PrevPowerOfTwo(int num)
{
    if (num <= 0) return 1;
    return (FillLower(num) >> 1) + 1;
}

IsPowerOfTwo

numが2の冪であるかを返します。 ↑

public static bool IsPowerOfTwo(this int num) => num > 0 && (num & (num - 1)) == 0;
