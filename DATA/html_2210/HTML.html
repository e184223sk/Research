UnityでMonaparty（その１）

C#WebGLUnityMonacoinMonaparty

たいそうなタイトルですね、、、

気づいたら2019年はMonaparty(Monacoin)で遊んでいたのでそのまとめを書いてみます。

はじめに

まず最初にモナコインの開発、維持に携わっている方々に感謝を申し上げます。また、イラストやグッズ開発、イベントの運営などの様々な活動をされている皆様、日々ナイスです。

MonapartyのAPIを使って何かつくってみる

Monapartyには様々なAPIがあり、APIを使えば様々なデータを取ってくることができます。

たとえば特定のアドレスが持っているトークンの種類や数量、トークンの発行や送金も可能です。（もっと知りたい方はこちらから　https://counterparty.io/docs/api/）
それらを使い出来たものが

『毎日のトランザクション数に応じてモナーが成長する謎のもの( ´∀`)』

です。

つくりかた

UnityではC#とJavascriptが使えます。
自分はC#を最初のポケモンに選んだのでC#でやっていますが、Javascriptの方が情報やライブラリが豊富でやりやすい気がします（自分の実力もありますが）

以下コード

APIをリクエスト.cs
void Start()
    {
        Monapa();
    }

public void Monapa()
    {
        string url = "リクエスト先APIサーバー";

        RequestData requestData = new RequestData();
        string json = JsonUtility.ToJson(requestData);

        StartCoroutine(Method(url, json));

        IEnumerator Method(string url, string bodyJsonString)
        {
            UnityWebRequest postRequest = new UnityWebRequest(url, "POST");
            byte[] bodyRaw = Encoding.UTF8.GetBytes(bodyJsonString);
            postRequest.uploadHandler = (UploadHandler)new UploadHandlerRaw(bodyRaw);
            postRequest.downloadHandler = (DownloadHandler)new DownloadHandlerBuffer();
            postRequest.SetRequestHeader("Content-Type", "application/json");

            yield return postRequest.SendWebRequest();

            ///データを受け取る.csの中にデータを入れてます
            InfoData infoData = new InfoData();
            infoData = JsonUtility.FromJson<InfoData>(postRequest.downloadHandler.text);

            if (postRequest.isHttpError || postRequest.isNetworkError)
            {
                Debug.Log(postRequest.error);
            }
            else
            {
　　　　　　　   ///取得したデータがズラッと表示されると思います
                Debug.Log(postRequest.downloadHandler.text);
　　　　　　　   ///ひとつずつ取り出すには
　　　　　　　   Debug.Log(infoData.result[0].asset);
            }
         }
    }
