【Unity3D】オリジナルゲーム「逆回転」

C#Unity3DUnityゲーム制作ゲーム開発

はじめに

こんにちは。
今回はUnityで作ったゲームに関する記事です。
私はあまり、3Dゲームは作らないのですが、この「逆回転」というゲームは回転することに重点を置いた3Dゲームになります。

完成するとこんな感じに動きます！

15個のCubeが回転しているのですが、ランダムで1つだけ他とは異なる回転をしているCubeがあります。
そのCubeをクリックで選択していき、正解すると得点が1上がり、不正解だと得点が1下がるようになっています。
5点取得するとゲーム終了です。
左下にタイマーがあるのですが、なるべく早く5点取得し、時間で競うゲームとなっています。

環境

  • Windows 10 home
  • Unity 2018.2.15f1

「逆回転」の制作

一からすべて解説するわけではないので、あくまでも一例として参考にしてみてください。

1.Hierarchy

まず、完成版のHierarchyは上図のような状態になっています。

2.Cubeのprefab化

Cubeを１つ作ったら、プレハブ化します。
プレハブ化したCube1にAdd ComponentでEvent Triggerを追加すると、Inspectorは上図のようになるはずです。

そしてこれを15個作っている感じです。
先ほどのEvent TriggerにはPointer Clickを追加して、後ほど説明する、処理が書かれているスクリプトがついているGameControllerというオブジェクトを選び、各それぞれ、対応するメソッドを選択すれば大丈夫です。

このEvent Triggerのクリックについてですが、うまくいかない場合は、こちらを参考にしてみてください。
→【Unity】マウスクリックで3Dオブジェクトを消すまで苦労した話

3.GameControlleroオブジェクト

空のオブジェクトを作り、GameController.csというスクリプトを追加します。

GameControllerオブジェクトのInspectorを上図のようになっています。
スクリプト内でpublic変数で宣言してあげたものの、対応する各それぞれをここに突っ込んであげてる感じですね。

スクリプトの内容は以下の通りです。

GameController.cs

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
using UnityEngine.SceneManagement;

public class GameController : MonoBehaviour{

    public GameObject[] cube;
    public Text scoreText;
    public Text timeText;
    public Text gamaClearText;
    public Button resetButton;

    private int rdnum;
    private int score=0;
    private float time;
    private bool gameflag;

    // Use this for initialization
    void Start () {
        gamaClearText.gameObject.SetActive (false);
        resetButton.gameObject.SetActive (false);

        rdnum = Random.Range (0, cube.Length);
        scoreText.text = "得 点 : ";
        gameflag = true;
    }


    // Update is called once per frame
    void Update () {

        if(gameflag){
            for(int i=0;i<cube.Length;i++){
                if (i != rdnum) {
                    cube [i].gameObject.transform.Rotate (3, 0, 3);
                } else {
                    cube [i].gameObject.transform.Rotate (-3, 0, -3);
                }
            }

            time += Time.deltaTime;
            timeText.text = time.ToString("f2");
            scoreText.text = "得 点 : " + score;

            if(score == 5){
                gameflag = false;
                gamaClearText.gameObject.SetActive (true);
                resetButton.gameObject.SetActive (true);
            }
        }
    }

    private void CorrectAnswer(){
        Debug.Log ("正解！！");
        rdnum = Random.Range (0, cube.Length);
        score++;
    }

    private void WrongAnswer(){
        Debug.Log ("残念！！");
        score--;
    }

    public void ResetButtonClick(){
        SceneManager.LoadScene ("main");
    }

    public void OnClickCube1(){
        if (rdnum == 0) {
            CorrectAnswer ();
        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube2(){
        if (rdnum == 1) {
            CorrectAnswer ();
        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube3(){
        if (rdnum == 2) {
            CorrectAnswer ();
        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube4(){
        if (rdnum == 3) {
            CorrectAnswer ();
        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube5(){
        if (rdnum == 4) {
            CorrectAnswer ();
        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube6(){
        if (rdnum == 5) {

        } else {
            WrongAnswer ();
        }
    }

    public void OnClickCube7(){
        if (rdnum == 6) {

            WrongAnswer ();
        }
    }

    public void OnClickCube8(){

        }
    }

    public void OnClickCube9(){

    }

    public void OnClickCube10(){

    }

5.その他、注意

シーン切り替え時にシーンが暗くなる問題で、その対処法としては、
Window → Rendering → Lighting Settingsから、Auto Generateのチェックを外し、Generate Lightingをクリックすれば、この問題は解決できるはずです。
