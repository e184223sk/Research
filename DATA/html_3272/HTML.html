WebアプリでJPGファイルをハンドリング

C#ASP.NETメモ

背景

通常、次のようなURLをブラウザでクリックすると、直接、画像ファイルをGETするため、
自作したWEBアプリでハンドリングできない。
画像ファイルのGETメソッドをWEBアプリで捕まえて、特定のフォルダにあるファイルを返すようにしたかったので、その方法について学習した。

環境

  • Visual Studio 2017
  • asp.net

問題

大きな問題は、Webアプリ(asp.Net)に関して初心者ということである。
プログラムを作るにあたって、次の二点が理解できなかった。

  • ルーティング

    リクエストを受けてコントローラーのメソッドを呼び出させる方法が分からない。

    例えば、http://hoge/test.jpg というリクエストが来た時に、hogeController.GetImage()を実行させる方法である。

  • 画像データをクライアントに渡す方法が分からない

分かったこと

インターネットの情報と同僚のコードから学んだ結果、次のような処理を用意すれば解決できることが分かった。

  • 指定URLのJEGをハンドリングすることをWebConfig.xmlに記述する。
  • 指定URLをハンドリングするコードを作成する。
    • アプリ起動時にそのコードを呼び出す。（例えばGlobal.asaxのApplication_Start）
  • バイナリを返すのはController.Fileを使えばよい

サンプル

<system.webServer>
    <handlers>
      <remove name="ExtensionlessUrlHandler-Integrated-4.0" />
      <remove name="OPTIONSVerbHandler" />
      <remove name="TRACEVerbHandler" />
      <add name="ExtensionlessUrlHandler-Integrated-4.0" path="*." verb="*" type="System.Web.Handlers.TransferRequestHandler" preCondition="integratedMode,runtimeVersionv4.0" />
      <!-- hoge/*.jpg のURLをWeb Appでハンドリングすることを設定 -->
      <add name="JpgFileHandler" path="hoge/*.jpg" verb="GET" type="System.Web.Handlers.TransferRequestHandler" preCondition="integratedMode,runtimeVersionv4.0" />
    </handlers>
  </system.webServer>

        protected void Application_Start()
        {
            // ルートを登録
            RouteConfig.RegisterRoutes(RouteTable.Routes);
        }

        public static void RegisterRoutes(RouteCollection routes)
        {
            string imageDir = TurfConfig_SynergyRecorder.Instance.SvidImageDirName.ConfigValue;

            routes.IgnoreRoute("{resource}.axd/{*pathInfo}");

            routes.RouteExistingFiles = true;

            // jpeg指定で呼び出されるコントローラーを設定
            routes.MapRoute(
                name: "hoge",
                url: "hoge/{filename}.jpg",// ハンドリング対象URL
                defaults: new { /*Controller Name*/controller = "Hoge", /*呼び出されるメソッド*/action = "GetImage" }
            );
        }
    }

    public class hogeController : Controller
    {
        public ActionResult GetImage()
        {
            byte [] image = LoadJpeg();
            return this.File(image, "image/jpeg");
        }
