【Unity】波のTerrainを作る♪

C#画像処理UnityterrainOculusGo

これって、案外簡単なのかなということでやってみた。
どうも参考の国土地理院のが失敗、さらにノイズ乗っけるとかというのはちょっとウワンにはむつかしかったので、ここはマンデルブロのPGをまねてやってみた。

【参考】
①フラクタル地形自動生成
②Unityで国土地理院のデータを使って地形を作る
参考①はやはり難しすぎてまねできない、参考②はマネできたがどうも結果が今一つ。一応こちらは全てのBMPが画像をとりあえず地形にできたので、面白いとは思いましたが、国土地理院の地図データ。。特に山岳データは穴ぼこだらけ、文字が窪んじゃう、等高線もダメそう。しかし人の画像の地形が描けたのは面白かった？？
※ただし、あのJavascriptツールはOculusGoと相性悪くて作成後に消去する手間が必要でした

ということで、自前で海の波位作れそうということでやってみました。

やったこと

（１）乱数で埋めつくす
（２）三角関数＋乱数でやってみる
（３）OculusGoで見る

（１）乱数で埋めつくす

アイディアはあのマンデルブロのプログラムの改変でできるはずということで、以下のようなコード書きました。
※主要な部分は以下のとおり

    float Calculate(double X, double Y)
    {
        float s= (float)Math.Sin(X*5);
        float s1 = 1 + (float)0.05 *UnityEngine.Random.value + (float)0.05 *(float)s;
        //float s1 = UnityEngine.Random.value;
        return s1;
    }
    IEnumerator UpdateTerrainCoroutine()
    {
        var heights = new float[terrainData.heightmapWidth, terrainData.heightmapHeight];
        while (true)
        { 
            renderArea *= 0.98;
            var startX = baseX - renderArea / 2.0;
            var startY = baseY - renderArea / 2.0;
            var renderAreaPerWidth = renderArea / terrainData.heightmapWidth;
            var renderAreaPerHeight = renderArea / terrainData.heightmapHeight;
            for (int x = 0; x < terrainData.heightmapWidth; x++)
            {
                for (int y = 0; y < terrainData.heightmapHeight; y++)
                {
                    heights[x, y] = Calculate(
                        startX + x * renderAreaPerWidth,
                        startY + y * renderAreaPerHeight);
                    heights[x, y] = heights[x, y]/ 2;
                }
            }
            terrainData.SetHeights(0, 0, heights);
            yield return null;
        }
    }
}

思惑では延々波が続くはずですが、一定の時間で波の周期が長くなってしまいます。
※X,Yをどこかで初期化したい、whileを工夫すればだけど今日はそこまでいきませんでした
まずは、float s1 = UnityEngine.Random.value;の方で動かすと乱数で埋めつくせます。
どんな感じかというと、Terrainの描画高さを２とかにすると、これが案外波っぽい。。。


これ近くで見るとやはり乱数というか細かい凸凹なんだけど、Unityが適当に平均化してこう見せてくれるんですね。

（２）三角関数＋乱数でやってみる

そして本命の三角関数。。これ実は最初は矩形にしか出なくて苦労しました。
ということで、

float s1 = 1 + (float)0.05 *UnityEngine.Random.value + (float)0.05 *(float)s;

で解決しました。

。。。あまり変わらないけど、以下動作中

そしてだんだん周期が長くなる。。。これくらいが続いてほしいけど。。

（３）OculusGoで見る

早速、。。こんな感じです。
一応、前回のFirstPersonControllerを使っているので回せます。

【Unity】波のイメージを作ってみる♪

※画像をクリックするとYouTube動画につながります

まとめ

・Terrainで波イメージで動く海を作成した
・Startで同じことをやると、乱数の大地や波（乱数込み）の大地の作成ができる
　これを時間とともに動かすのもいいかもね

・今のコードだとだんだん周期が長くなりやがて乱数のみになってしまう
・大地に対しては適当にいろいろな処理をすればいろいろ出来そうである
・国土地理院の地図は今のところ針の山しか出来ていない
　※ただし、いろいろなBMP画像で大地を作成できるので楽しい
　　OculusGoでは、作成後毎回Jsを削除する必要がある
