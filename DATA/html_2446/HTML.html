C#の開発環境をVisual Studio Community 2019で構築しよう

C#初心者VisualStudio2019

はじめに

プログラミング言語C#を使って様々なタイプのアプリケーション（Webアプリ、デスクトップアプリ）を開発するための環境を構築します。今回はVisual Studio Community 2019をインストールして環境を構築します。

C#とは

Microsoftが開発したプログラミング言語です。Microsoftのフレームワーク（OSの機能を操作するための機能の集合体、ライブラリ）「.NET Framework」の一部です。一昔前はVB.NETと双璧を成していましたが、近年ではC#の一強状態です。
詳細はWikipediaを参照してください。

Visual Studio Community 2019とは

Microsoftが提供する統合開発環境（IDE）です。C#で開発したりビルドしたり、Azureにデプロイしたり、チームで開発するための便利機能が提供されていたりします。
C#以外にもVB.NETやPythonの開発もサポートしています。
詳細は公式サイトを参照してください。
Visual Studioは数年に一度バージョンアップされており、現在は2019が最新バージョンになります。また、エディションはCommunity、Professional、Enterpriseの3つがあり、Communityは無料で使えるエディションになります。
今回は無料で使えるCommunityをインストールしていきます。

Visual Studio Community 2019をインストールする

Visual Studio Community 2019をインストールします。
インストールの作業では、大量データのダウンロードとそれなりの時間がかかりますのでご注意ください。

1. Visual Studioのサイトからインストーラをダウンロードします。
Visual Studioの公式サイトに移動し、［Visual Studioのダウンロード］をクリックするとエディションが表示されますので、［Community 2019］を選択すると、ダウンローダ実行されます。


2. インストールを実行します。
インストーラを起動してダウンロードが実行されます。その後、Visual Studioのインストーラが起動されるので、自分が利用したいものにチェックして、インストールを実行します。
今回は、C#でWebアプリケーション、デスクトップアプリケーションを作成したいので、以下のものにチェックします。

選択する項目

  • ASP.NETとWeb開発
  • .NETデスクトップ開発

開発環境のディスク容量に余裕があれば、とりあえず全部選択するのもいいですし、必要であれば後ほど追加インストールすることも可能です。
準備が整えば、右下の［変更］をクリックします。

Visual StudioでC#のコンソールアプリケーションを作ってプログラムを実行する

インストールしたVisual Studio Community 2019でコンソールアプリケーションを作成し、プログラミングを実行します。

1. Visual Studioを起動します。
Visual Studioを起動します。初回起動時にはMicrosoftアカウントのログインを求められます。ログインしてもいいですし、スキップしても構いません。
起動が完了すると以下の様な画面が表示されます。

2. 新しいプロジェクトを作成します。
今回の実行用にプロジェクト（アプリケーションの開発や実行に必要なファイルの集合体）を作成するために、［新しいプロジェクトの作成］をクリックします。

上にあるテキストボックスに「コンソール」と入力して検索するとコンソールアプリケーションのプロジェクトテンプレートが表示されます。ここで、C#のコンソールアプリケーションを選択し、［次へ］をクリックします。

プロジェクト名、場所、ソリューション名に任意の値を入力して［次へ］をクリックします。

ターゲットフレームワークに任意の値を選択して［作成］をクリックします。
今後のサポート状況等を考えると.NET Core 3.xを選択するのがベターです。

プロジェクトが作成されてVisual Studio Community 2019が表示されます。

3. プログラムを実行します。
初期値で以下の様なデバッグコンソール画面に「Hello World!」というメッセージを表示するソースコードが記載されていますので、そのまま利用します。

Program.cs
class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine(""Hello World!"");
    }
}

（色々と実行する方法はありますが、）キーボードのF5をクリックします。
デバッグコンソール画面が表示され、そこに「Hello World!」という文字が表示されます。
これで環境構築は完了です。

さいごに

インストールの作業自体はこれで完了です。これで、どんなアプリケーションも開発することが可能になります。
みなさんもこの手順に従って、自分のパソコンに開発環境を作って、プログラミングライフをエンジョイしてください！
