シャッフル再生のアルゴリズムを考える

C#アルゴリズム

概要

音楽プレイヤーのシャッフル再生やtwitterのbotのように「ランダムに一つずつ選択したい」という機能を作るときの「ランダム」はどう実現するのが良いかということがふと気になったので実装してみました。

単純にリストからランダムに選ぶと同じものが連続して選ばれるといったような現象が発生します。参考
これは自然なシャッフル再生ではなくなってしまいます。

それぞれの要素に重みを付けたランダム選択を行い選択されたら重みを0にする。選択されなければ重み付けを回復させていく という手法で自然なシャッフル再生を実現したいと思います。

重み付けランダムは重みをつけてランダムに何か出したい のようにして実現できそうです。

実装

C#で上記の機能を実装しました。

  • 全要素の重みを要素数で初期化
  • 選出されたら重みを0にする
  • 重みが初期値(要素数)でなければ他の要素が選出されるたびにインクリメント

として重み付けを行っています。

ここの重み付けの仕方がアルゴリズムの良さに直結するかと思うので、市販のプレイヤーアプリではもっと複雑なアルゴリズムを採用しているかもしれません。

using System;
using System.Collections.Generic;
using System.Linq;

public class WeightRandom<T>
{
    private readonly Random random;

    private T[] Items { get; }
    private int[] Weights { get; }

    public WeightRandom(IEnumerable<T> items) : this(items, new Random()) { }
    public WeightRandom(IEnumerable<T> items, Random random)
    {
        if (!items.Any()) throw new Exception("空はだめ");

        this.random = random;
        this.Items = items.ToArray();
        this.Weights = Enumerable.Repeat(this.Items.Length, this.Items.Length).ToArray();
    }

    private void UpdateWeights(int targetIndex)
    {
        for (int i = 0; i < this.Weights.Length; i++)
        {
            if (this.Weights[i] < this.Weights.Length)
                this.Weights[i]++;
        }
        this.Weights[targetIndex] = 0;
    }
    public T GetNext()
    {
        var allWeight = this.Weights.Sum();
        var randomValue = this.random.Next(allWeight);

        int index;
        int weightSum = 0;
        for (index = 0; index < this.Items.Length; index++)
        {
            weightSum += this.Weights[index];
            if (randomValue < weightSum)
                break;
        }

        this.UpdateWeights(index);
        return this.Items[index];
    }
}

IEnumerable<int> ToDiff(IEnumerable<int> indices)
{
    var arr = indices.ToArray();
    if (arr.Length < 2)
        yield break;

    for (int i = 1; i < arr.Length; i++)
    {
        yield return arr[i] - arr[i - 1];
    }
}

    this.Weights[targetIndex] = 0;

結論

  • 選出されたら重みを0にする
  • 選出されなかったら重みをインクリメントする

この2つの条件で重み付けしたうえでランダム選出することでいい感じにシャッフルができそうでした。
