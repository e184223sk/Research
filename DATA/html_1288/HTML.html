【C#】テキストファイルを任意の行を読み込む

C#

自己紹介

こんにちは。tetraです。
最近C#の学習を始めました。C#歴3ヶ月目の新卒エンジニアです。
任意の行で表示したくてもテキストファイルを上から順に読む処理が出てムカついたので新しく作成します。

ファイル読み込み

さっそく書いていきます。
１行目を出力しています。

sample.cs
using System;
using System.IO;
using System.Linq;

class sample
{
    static void Main()
    {
        // skipのパラメーターをいじると任意の行を指定できる
        int n = 0;
        string line = File.ReadLines("./sample.txt").Skip(n).First();
        Console.WriteLine(line);
    }
}
