この記事で分からなかったら参考にしてくださいね！今回は例としてこんな感じのクラスを用意しました。では配列"UserDATA[]"を文字列に変換していきます。
各自のクラスに対応できるようにジェネリックを使ってます。※なお、よく使うstring型・int,float,double型しか書いていないので他の型がクラスに入っている場合各自対応してください。(書いていないけどbool,byteなどは一応動く)例えばUnity上のVector3・Colorなどはこのコードだけではそのまま保存できません。
※ Color型 → int R, int G, int B, int A に変換
※ Vector3型　→ float x ,float y, float z に変換してから等...では今度は文字列からクラスに戻していきます。今回の例"UserDataクラス"の配列を文字列変換してみるとこんな感じになります。
※"tagName"に「P」を指定しています。

しっかり変換されて保存できてますね。アプリでこのまま端末に平文保存するのはあまりよくないです。
もっと良い保存方法があるはずなのであくまで参考程度にどうぞ。
